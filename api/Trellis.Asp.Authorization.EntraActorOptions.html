Remarks

Default behavior:

  • Id is set from the oid (Object ID) claim.
  • Permissions are mapped from claims of type roles.
  • ForbiddenPermissions is empty.
  • Attributes includes tid, preferred_username, azp, azpacr, acrs, ip_address, and mfa (see ActorAttributes).

Override any delegate to customize mapping. All delegates receive the authenticated user's claims as IEnumerable<T> of Claim. MapAttributes additionally receives the HttpContext for request-level context (e.g., IP address).
