ToPartialOrOkActionResultAsync<TIn, TOut>(Task<Result<TIn>>, ControllerBase, Func<TIn, ContentRangeAndData<TOut>>)

If result is in success state, returns Okay (200) or Partial (206) status code based on the ContentRangeHeaderValue.

If result is in failed state, returns a failed ObjectResult based on mapping ToErrorActionResult<TValue>(Error, ControllerBase).
