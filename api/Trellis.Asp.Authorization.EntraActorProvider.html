Remarks

Register as scoped in DI via AddEntraActorProvider(IServiceCollection, Action<EntraActorOptions>?). Mapping behavior is controlled by EntraActorOptions.

This provider assumes authentication has already occurred (e.g., via AddMicrosoftIdentityWebApi). It throws InvalidOperationException if no authenticated user exists.
