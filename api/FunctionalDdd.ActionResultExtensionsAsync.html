ToPartialOrOkActionResultAsync<TIn, TOut>(Task<Result<TIn>>, ControllerBase, Func<TIn, ContentRangeHeaderValue>, Func<TIn, TOut>)

If Result<TValue> is in success state this extension method returns

  • Partial Content (206) with header ContentRangeHeaderValue for partial results
  • Okay (200) status for complete results.
Otherwise it returns the error code corresponding to the failure error object.
